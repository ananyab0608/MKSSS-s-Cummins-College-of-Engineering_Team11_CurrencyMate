Problem Statement we worked on..

Create a dashboard that enables users to analyze exchange rates between two currencies over various timeframes, including weekly, monthly, quarterly, and yearly charts.

It should allow users to track currency trends and highlights the peak (highest) and lowest exchange rates during the selected period.

Additionally, users have the option to print the chart data for easy reference. The base currency is always set to USD, with the second currency variable, such as USD/INR, USD/GBP, or USD/EUR.
